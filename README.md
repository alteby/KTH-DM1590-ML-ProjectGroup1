# KTH-DM1590-ML-ProjectGroup1
Develop a machine learning system that can distinguish which animal is being shown in a picture.  We will use supervised learning and train a kNN-model to classify and find patterns in the animals from our dataset. We will also use unsupervised learning, PCA for dimensionality reduction and visualization and k-means clustering. We are using a labeled dataset with animal images from Kaggle. Each image belongs to an animal class. 

Dataset: https://www.kaggle.com/datasets/alessiocorrado99/animals10/data
